# Convertor
A combine tool set of XML/Json format/diff, Base64/URL en/decode etc... 
Save engineers' time from endless manual en/decode/diff job.

## Contribute
1. `git clone`
2. `yarn` or `yarn install`
3. `yarn build`
4. `yarn lint` 
5. `yarn test`
